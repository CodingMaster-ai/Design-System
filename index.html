Specific Text Element Styles

Main Heading/Title (h1)

  • Font Size: 40px
  • Font Family: PT Serif
  • Font Weight: Normal
  • Text Decoration: Underline
  • Background Color: Yellow
  • Highlight or Underline (Sometimes Both)

Sub Heading (h2)

  • Font Family: Lato
  • Font Size: 25px
  • Font Weight: Bold

Paragraph (p)

  • Font Family: Lato
  • Font Size: 18px
  • Line Height: 1.5

Lists/If On The Same Page (ul & li)

  • Font Family: Lato
  • Font Size: 20px
  • Margin Bottom: 20px (Roughly)
  • Margin: 0 auto (If Possible)
  • List Style Type: Square, Circle, or Normal (Depends)

Once again these are prototype ideas and will change and be solidified once I figure out what styles work the best for the type of website I'm building.

Additional Styles (If Possible)

  • Background Color: for element contrast
  • Border or Border-Style: to give elements a unique appearance and emphasis
  • Border-Box: for appropriate sizing
  • Underline or Bold: more emphasis
  • Inline-Style: to align elements side-by-side
  • Positioning: in order to align the elements with others on the page for a cohesive flow
  • Padding and Margin: in order to properly separate and display the elements
  • Buttons, Links, and Navigation: to allow for easy interactivity for users on the site

There are more styles that I could include but these are a few that will be used quite frequently at the moment but I could change my mind as I improve my skills as a Front-end Developer. Therefore I will be able to incorporate more complex and useful styles that can make my page look and function in a more advanced manner.
